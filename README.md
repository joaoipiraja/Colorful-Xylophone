# Colorful-Xylophone
Colorful Xylophone IOS APP using Swift 5.1
### Objective 📝
Learn how to use  ``` AVAudioPlayer``` and ``` UIView.animate``` with ``` DispatchQueue.main.asyncAfter``` to create a button animation.
